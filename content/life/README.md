# Life Lab 记录

这里放 Life Lab 内容：日常记录、记忆片段、观察和短笔记。

推荐流程：

1. 复制 `_template.md`。
2. 改成日期开头的文件名，例如 `2026-07-09-morning-light.md`。
3. 用 Markdown 写记录。
4. 相关图片或附件放到 `content/life/assets/`。
5. 准备发布时，把 `status` 改成 `published`。

建议文件名格式：

```text
YYYY-MM-DD-life-slug.md
```

Life Lab 常用字段：

- `title`: 生活记录标题。
- `date`: 记录日期。
- `time`: 记录时间，例如 `08:30`。
- `status`: 发布状态，`published` 才会显示在列表里。
- `mood`: 心情或氛围，例如 `安静`、`松弛`、`明亮`。
- `place`: 地点，例如 `家里`、`工作室`。
- `tags`: 标签数组，例如 `["日常", "光", "房间"]`。
- `summary`: 列表摘要。
- `image`: 图片文件名，图片放在 `content/life/assets/`。
